# LittleGrow UI 重构设计说明 (UI Overhaul Design Spec)

**日期**: 2026-04-09  
**目标**: 将长呀长 (LittleGrow) App 的 UI 彻底重构为精致、温润、具有成熟商业级质感的应用。
**整体风格**: 精致温润（柔和马卡龙/莫兰迪色系、大圆角、轻盈阴影、强信息层级）。

## 1. 全局导航与交互架构 (Architecture & Navigation)
- **底层架构**: 使用带有柔和阴影与高斯模糊感的 `Scaffold` 作为页面基础结构。
- **底部导航栏 (NavigationBar)**: 
  - 移除 Material 3 默认的强高亮背景。
  - 选中状态：使用主题色（Primary）图标与文字，增加轻微缩放动画反馈。
  - 未选中状态：柔和的灰色/次要色。
- **全局悬浮按钮 (FAB)**: 
  - 驻留在首页及主要数据页右下角。
  - 样式：主题色圆角矩形（16dp 圆角），内部包含一个精致的“添加”图标。
  - 交互：点击唤出底部面板（ModalBottomSheet），包含所有快捷记录入口（喂养、排泄、睡眠等），实现完美的单手拇指操作。

## 2. 首页看板重构 (Home Dashboard)
- **沉浸式顶部区域 (Hero Header)**: 
  - 替代传统的系统 `TopAppBar`。
  - 背景：使用柔和的马卡龙色渐变底板或卡片。
  - 内容：左侧显示宝宝头像（带白色描边与浅阴影的圆形/圆角矩形），右侧大字号（HeadlineMedium/Bold）显示宝宝姓名、日龄/月龄信息，辅以温和的问候语。
- **今日数据网格 (Summary Grid)**: 
  - 使用 2x2 或流式布局展示核心数据（如：今日喂奶次数、排泄次数、睡眠总时长）。
  - 卡片样式：`ElevatedCard`，背景色为 `SecondaryContainer` 或纯白（带浅色背景时），24dp 大圆角，极其柔和的弥散阴影。
  - 图标：使用重新绘制的单色或双色精致图标。
- **最近动态流 (Recent Timeline)**: 
  - 移除边框和厚重背景。
  - 使用留白和极细分割线，列表项左侧带类别小图标气泡，右侧显示时间和摘要。

## 3. 记录表单交互优化 (Forms & Interactions)
- **半屏面板 (ModalBottomSheet)**: 
  - 所有的快捷记录动作都在底部弹出的面板中完成，无需跳转全屏新页面，维持上下文感。
- **超大触控区 (Large Tap Targets)**: 
  - 便便颜色/性状等选择器：放弃复选框，改用大尺寸的色块按钮或图标按钮（Radio Button 样式），点击自带缩放动效。
- **快捷输入 (Quick Inputs)**: 
  - 奶量等数字输入增加 `[+10]`, `[+30]` 等快捷增量按钮，大幅减少系统键盘的使用。

## 4. 主题与视觉系统 (Theme & Typography)
- **色彩调优**: 
  - 维持现有核心马卡龙色调（青绿、粉蓝、浅黄），但增加对比度。
  - 背景色：使用极浅奶油色（如 `#FFFBF6`，暗色模式下为深灰），使纯白/纯黑卡片产生“悬浮感”。
- **排版 (Typography)**: 
  - 标题（Headline/Title）字重加粗，正文保持常规。
  - 优化行高与字间距，使得阅读数据时呼吸感更强。
- **圆角与阴影 (Shapes & Shadows)**: 
  - 全局统一使用大圆角（卡片 24dp，按钮 16dp）。
  - 使用自定义 `Modifier.shadow` 参数（降低 alpha，增加 blur radius），替代系统默认生硬的黑色阴影。
- **全新图标绘制 (Custom Icons)**: 
  - 为所有分类（母乳、奶瓶、尿布、睡眠等）重新绘制一套风格统一的线条或面性图标，圆润无锐角。

## 5. 验收标准
1. UI 元素不再有“原生拼凑感”，整体感觉统一且高级。
2. 黑暗模式下的对比度适宜，不刺眼。
3. 从首页发起的常用记录动作，点击步骤不超过 2-3 次，且完全可单手完成。
4. 所有自定义图标到位。
